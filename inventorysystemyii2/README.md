# 🧾 Inventory Management System (Yii2 + SQLite)

This is a simple **Inventory CRUD system** built with the **Yii2 Framework** and **SQLite** database.

## 📁 Features

- ✅ Add new products
- ✅ View product list
- ✅ Update existing products
- ✅ Delete products

## 🛠 Tech Stack

- PHP (Yii2 Framework)
- SQLite
- Bootstrap (default Yii2 styling)
